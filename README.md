- **弥补Windows 11 不显示秒数的小工具**

## 说明
- 无主程序界面， 无设置界面， 双击启动即可生效（仅限windows 11）

### 如何设置开机自启？
- 程序除了显秒未添加任何功能， 所以需要手动添加开机自启
    -  右键ShowSeconds.exe创建快捷方式
    -  win + R打开运行界面， 输入： **shell:startup** 点击回车即可打开启动目录
    -  将创建的快捷方式拖进启动目录即可
### 如何退出？
- 打开任务管理器， 找到 ShowSeconds, 结束进程即可
